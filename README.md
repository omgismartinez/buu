## **Run Locally**

Clone the project

```bash
  git clone https://github.com/omgismartinez/buu.git
```

Go to the project directory

```bash
  cd buu
```

Install dependencies

```bash
  npm install | yarn
```

Start the server

```bash
  npm run dev | yarn dev
```
